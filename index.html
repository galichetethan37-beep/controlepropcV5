Guide de démarrage
1
Lancez l'agent Python — double-cliquez start-agent.bat (Windows) ou ./start-agent.sh (Mac/Linux). Il installe automatiquement les dépendances.
2
Cliquez "Partager l'écran" — choisissez un onglet, une fenêtre ou l'écran entier dans Chrome/Edge
3
Sur votre smartphone, ouvrez cette même URL et entrez le code à 6 caractères
4
Contrôlez votre PC — tap = clic gauche, appui long = clic droit, double-tap = double clic, deux doigts = défilement
⚡
L'agent Python (remotelink-agent.py) doit tourner sur le PC pour le contrôle réel. Il écoute sur localhost:8765 et utilise PyAutoGUI pour déplacer la souris et simuler les clics au niveau système d'exploitation.
